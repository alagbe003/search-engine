# Deployment Guide\n\n## GitHub Pages (Frontend)\n\n### Prerequisites\n- GitHub account\n- Repository access\n\n### Steps\n\n1. Update `package.json` in frontend:\n```json\n{\n  "homepage": "https://alagbe003.github.io/search-engine",\n  "scripts": {\n    "predeploy": "npm run build",\n    "deploy": "gh-pages -d build"\n  }\n}\n```\n\n2. Install gh-pages:\n```bash\ncd frontend\nnpm install --save-dev gh-pages\n```\n\n3. Deploy:\n```bash\nnpm run deploy\n```\n\n## Heroku Deployment (Backend)\n\n### Prerequisites\n- Heroku account\n- Heroku CLI installed\n\n### Steps\n\n1. Login to Heroku:\n```bash\nheroku login\n```\n\n2. Create app:\n```bash\nheroku create search-engine-api\n```\n\n3. Add MongoDB Atlas:\n```bash\nheroku addons:create mongolab\n```\n\n4. Set environment variables:\n```bash\nheroku config:set NODE_ENV=production\nheroku config:set CORS_ORIGIN=your-frontend-url\n```\n\n5. Deploy:\n```bash\ngit push heroku main\n```\n\n## Docker Hub\n\n### Build and Push Images\n\n1. Build images:\n```bash\ndocker-compose build\n```\n\n2. Tag images:\n```bash\ndocker tag search-engine-backend:latest alagbe003/search-engine-backend:latest\ndocker tag search-engine-frontend:latest alagbe003/search-engine-frontend:latest\n```\n\n3. Push to Docker Hub:\n```bash\ndocker push alagbe003/search-engine-backend:latest\ndocker push alagbe003/search-engine-frontend:latest\n```\n\n## AWS Deployment\n\n### Using Elastic Beanstalk\n\n1. Install EB CLI:\n```bash\npip install awsebcli\n```\n\n2. Initialize:\n```bash\neb init -p node.js-18 search-engine\n```\n\n3. Create environment:\n```bash\neb create search-engine-env\n```\n\n4. Deploy:\n```bash\neb deploy\n```\n\n## Environment Variables for Production\n\n```\nNODE_ENV=production\nMONGO_URI=your-mongodb-atlas-uri\nCORS_ORIGIN=your-frontend-url\nSEARCH_API_KEY=your-production-key\nGOOGLE_API_KEY=your-google-api-key\nNEWS_API_KEY=your-news-api-key\n```\n
